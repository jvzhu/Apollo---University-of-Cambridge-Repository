# Code of Conduct

This repository documents academic research and collaboration. Anyone interacting here — via issues, pull requests, or discussion — is expected to:

- Communicate respectfully and in good faith.
- Give accurate attribution when referencing others' work.
- Raise disagreements or corrections constructively.

Reports of inappropriate conduct related to this repository can be made by opening an issue or contacting jiaqianvivienzhu@gmail.com directly.
