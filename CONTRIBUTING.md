# Contributing

This is a personal research repository. It is not currently open to external code contributions in the typical open-source sense, but collaborators on projects documented here (e.g. FigurativeAccess team members) are welcome to suggest corrections or additions via an issue or pull request.

## How to suggest a change

1. Open an issue describing the correction, addition, or concern.
2. If proposing a specific text change, feel free to open a pull request referencing the issue.
3. I'll review and merge or respond as time allows.

## Scope

Contributions relevant to this repository include:
- Corrections to bibliographic or source information in `data/sources.md`
- Notes or documentation related to the FigurativeAccess project or other Cambridge-affiliated work listed in the README

Contributions outside this scope (e.g. unrelated research areas) are better suited to a separate repository — feel free to open an issue to discuss.
